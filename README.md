# SkillNovus AI

## Overview

SkillNovus AI is an AI-powered learning assistant developed using Python, FastAPI, HTML, CSS, JavaScript, and Google Gemini API.

It helps students with:

- AI Tutor
- Coding Assistance
- Interview Preparation
- MCQ Generation
- Study Planning

## Technologies Used

### Frontend
- HTML
- CSS
- JavaScript

### Backend
- Python
- FastAPI

### AI Model
- Google Gemini API

## Features

### AI Tutor
Explains concepts in simple language.

### Coding Assistant
Generates and explains code.

### Interview Assistant
Provides interview questions and answers.

### MCQ Generator
Creates multiple-choice questions.

### Study Planner
Helps students organize learning schedules.

## How to Run

### Backend

```bash
python -m uvicorn app:app --reload
```

### Frontend

Open index.html using Live Server.
